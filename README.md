📚 Book Recommendation System

A Python-based web scraping project that extracts book data and provides recommendations based on user preferences.

🚀 Features
Scrapes book titles and ratings from a website
Converts ratings from text (e.g., "Three") to numeric values
Allows user to input minimum rating
Recommends a random book that meets the criteria
Handles cases where no books match
🧠 What I Learned
Web scraping using requests and BeautifulSoup
Parsing HTML structure
Working with dictionaries and lists
Building filtering and selection logic
Handling real-world edge cases
